# vue2zxerp

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

## 技术栈

+ Vue2 语法
  + data
  + methods
  + components
  + props
+ Vue Cli @3  脚手架
+ Vue router  路由库
  + router-view
  + Recursive
+ Element UI  组件库
+ Sass  CSS的预处理器
  + Sass-loader 加载器
+ axios 网络请求库
